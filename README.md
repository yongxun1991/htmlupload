# HTML Upload Repository

This repository contains various HTML files used for testing mobile slide shows, ad units, and utility tools like a Unix timestamp converter.

## Files

- `index.html`, `index2.html`, `index3.html`, `index4.html`: HTML files demonstrating mobile slide show ad formats.
- `unixtimeconverter.html`: A utility tool to convert between Date/Time and Unix Timestamps.
- `slideshowformobile.html`: Another slide show demonstration.
- `imasdkwithvpn.html`: Likely related to IMA SDK testing.
- `infoline-csv.html`: HTML file seemingly related to CSV info lines.
- `sample.xlsx`: A sample Excel file.

## Usage

To view the HTML files in this repository, you can use the provided `server.py` script to start a local HTTP server.

```bash
python3 server.py
```

Then open your browser and navigate to `http://localhost:8000`.
